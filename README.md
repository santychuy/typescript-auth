# Typescript Auth w/JWT

Basic example app to show the basics of JWT [(Info)](https://jwt.io/), and practice w/TypeScript.

## Getting Started

### Don't Forget

``` javascript
npm i -S
```

### NPM Commands

``` javascript
npm run start
npm run dev
```
